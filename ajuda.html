Ajuda - SIB

Bem-vindo à página de ajuda do SIB. Aqui você encontrará informações sobre a busca de termos no Pje.


Como realizar uma busca de termos:
  1. Abra o painel de busca do SIB.
  2. Informe o termo que deseja buscar no campo de busca.
  3. Selecione os filtros opcionais, se necessário.
  4. Clique no botão "Buscar" para iniciar a busca.

Dicas para busca eficiente:
  • Utilize aspas para buscar uma frase completa: "termo de busca".
  • Use operadores lógicos como AND, OR, NOT para refinar a busca.
  • Utilize os filtros para restringir a busca por data, tipo de documento, entre outros critérios.
  • Experimente diferentes termos de busca para obter resultados mais relevantes.

Se você tiver alguma dúvida adicional, entre em contato com nossa equipe de suporte.
